Tips

Set the model size during conversion (Scale parameter). The game won't scale it nicely by itself. For small decor objects you can use a very low scale like 0.06.

Cylinder Radius depends on your Scale. If you increase the model Scale, increase Cylinder Radius to keep wires visually proportional; if you decrease Scale, reduce the radius so wires don't look too thick.

Good starting point: try Scale = 2 and Cylinder Radius = 0.5, then adjust to taste.

Low‑poly is not mandatory, but heavy objects won't load nicely in the game. Aim for an exported .world under 500 KB.

To reduce complexity in Blender (optional):

  1. File → Import → STL
  2. Select the mesh → Modifiers → Add Modifier → Decimate
  3. Adjust Ratio (e.g. 0.1–0.5) to reduce triangles → Apply
  4. File → Export → STL
